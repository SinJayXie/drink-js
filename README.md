#`Drink-js`
``Http api server, has controller model middieware``

#`Node Version`
``node >= 16``

#`First install`
``yarn install && npm install``

#`Dev Service`
``yarn run dev && npm run dev``

#`build Service`
``yarn run build && npm run build``

Drink-js help from [[Github](https://github.com/SinJayXie/drink-js)]
